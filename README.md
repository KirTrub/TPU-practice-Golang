# Система управления базой данных "Больница"

Данный проект представляет собой веб-интерфейс для взаимодействия с базой данных медицинской организации. Система позволяет автоматизировать работу с пациентами, персоналом, отделениями и вести учет госпитализаций.

## О проекте

Проект состоит из двух частей:

### Backend
Написан на Go (Golang) с использованием PostgreSQL.

### Frontend
Написан на React с использованием TypeScript и библиотеки Mantine UI.

## Быстрый запуск

Все компоненты системы (база данных, сервер и клиент) настроены для работы в контейнерах.
Выполните команду в корневой директории проекта:

```bash
docker-compose up --build
```

После завершения сборки:

- Интерфейс будет доступен по адресу: http://localhost:5173  
- Backend работает на порту: 8080  

## Основные разделы и страницы

Приложение предоставляет следующие возможности:

### 1. Управление пациентами

- Просмотр полного списка пациентов  
- Форма добавления нового пациента (ФИО, пол, адрес, дата рождения)  
- Редактирование и удаление записей  

### 2. Учет персонала (Врачи)

- Ведение списка врачей с указанием их специализации и привязки к отделениям  
- Интеграция данных о врачах в процессы госпитализации  

### 3. Отделения и диагнозы

- Справочники для управления структурой больницы и классификацией болезней  
- Возможность добавления и актуализации наименований отделений и типов диагнозов  

### 4. Журнал госпитализаций

Основной рабочий инструмент для регистрации пациентов в стационаре  

- Выбор пациента, лечащего врача, отделения и диагноза из справочников  
- Указание дат начала и планируемого завершения лечения  

### 5. Аналитическая отчетность

Специализированная страница для генерации отчетов  

- Фильтрация данных по конкретному отделению и году  
- Расчет статистики: количество уникальных пациентов, минимальные и максимальные сроки лечения в разрезе врачей и диагнозов
